Um diesen Website zu anschauen braucht ihr die folgenden Schrite machen:

1.Xampp instalieren
2.Xampp starten und drin Apache und Mysql starten
3.Die "linker" sql Tabelle auf ihren SQL Database hochladen in PHPmyadmin
4. Mit Browser über Localhost sich auf die website verbinden
